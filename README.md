# EiN.js 
####A Simple Entity Component System####


## Installing
`jspm install EiN=github:SoftwareSyndicate/EiN@dev`

## Testing
`npm run test`

## Using
```javascript
import em from 'EiN'

var someEntity = new SomeEntity();
em.addEntity(sameEntity);
em.update(delta);
```
