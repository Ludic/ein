# EiN.js 
####A Simple Entity Component System####


## Installing

#### jspm

`jspm install ein=github:Ludic/ein`

#### webpack

`npm install github:Ludic/ein`

## Testing
`npm run test`

## Using
```javascript
import em from 'EiN'

var someEntity = new SomeEntity();
em.addEntity(sameEntity);
em.update(delta);
```
