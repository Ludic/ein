# EiN.js **A Simple Entity Component System**


## Installing
`jspm install EiN=github:SoftwareSyndicate/EiN@dev`

## Testing
`npm run test`

## Using
```javascript
import EiN as ein from 'EiN'

var someEntity = new SomeEntity();
ein.addEntity(sameEntity);
ein.update(delta);
```
